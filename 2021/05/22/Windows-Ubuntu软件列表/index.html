加粗代表常用的软件。

Windows

网络

  • Chrome：最好用的浏览器！

    • uBlock Origin：比AdBlock好用的广告拦截
    • SwitchyOmega：不可描述
    • 沙拉查词：划词翻译
    • TamperMonkey：运行脚本
    • setupVPN：不可描述
    • HoxxVPN：不可描述
    • FireShot：网页截图

  • ShadowsocksR：不可描述

工具

  • QQ：社交
  • 网易云音乐：音乐
  • Microsoft office：Windows下为数不多的好东西
  • shotcut：不知道好不好用的免费开源剪辑软件

Ubuntu 20.04

网络

  • Chrome：最好用的浏览器！
    • uBlock Origin：比AdBlock好用的广告拦截
    • SwitchyOmega：不可描述
    • 沙拉查词：划词翻译
    • TamperMonkey：运行脚本
    • setupVPN：不可描述
    • HoxxVPN：不可描述
    • FireShot：网页截图
  • net-tools：基础工具
  • electron-ssr：不可描述
  • ssh：传输文件
  • git：使用github必备。

工具

  • 搜狗输入法：良好的中文输入体验
  • gnome-tweaks：额外的设置。
  • zsh：好用的shell

  • gnome-shell-extensions：各种插件

    • topicons-plus：显示后台应用
    • desktop-icons-ng-ding：比20.04好用的桌面图标功能

  • QQ：社交

  • vim：好用的文本编辑器

    • youcompleteme：还算好用的补全

  • typora：编辑md文件

  • hexo：写博客
