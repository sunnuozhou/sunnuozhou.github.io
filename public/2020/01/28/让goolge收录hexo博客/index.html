让goolge收录hexo博客

Posted on 2020-01-28 Edited on 2020-01-30 In 教程 Views:

我发现在google上搜不到自己的博客，就研究了一下。

生成站点地图

先在自己的网站中生成站点地图，在博客目录下npm install hexo-generator-sitemap --save安装插件。

然后再配置文件中插入以下内容：

现在hexo g之后就可以在/public中发现sitemap.xml。

验证网站所有权

我选的是$google$推荐的验证方式——上传html。先在Google Search Console输入你的网址，选择继续，把$google$的html放到博客文件夹的public目录下，然后hexo d上传。

等上传完后开始验证就好了。

上传站点地图

在左侧选择站点地图一项，输入站点地图的位置，然后上传就好了。
